# Semantic Commit Emojis

Map the commit type to the corresponding emoji:

- feat: ✨
- fix: 🐛
- docs: 📝
- refactor: ♻️
- style: 🎨
- test: 🔬
- perf: ⚡
- hotfix: 🚑
- locale: 🌐
- ci: 👷
- chore: 🔧
- types: 🏷️

**Usage:** Place the emoji at the very beginning of the commit message followed by a space.
*Example:* ✨ feat: add new authentication flow
