# @adambullmer/ai-theme-monokai-soda

Monokai Soda theme for Gemini and other AI agents.

## Description

The `monokai-soda` theme provides a vibrant and high-contrast color scheme inspired by the popular Monokai Soda color palette. This package contains the theme definitions for use with various AI agent interfaces and development tools.

## Usage

To use this theme, follow the instructions for your specific tool or framework. Typically, this involves importing the theme's configuration and applying it to your interface.

```json
{
  "theme": "@adambullmer/ai-theme-monokai-soda"
}
```

## Features

- High contrast for better readability.
- Consistent color palette for a variety of syntax elements.
- Optimized for AI-assisted development workflows.
